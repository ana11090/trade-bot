# Project 2 - Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Prerequisites ✅

Make sure you have completed:
- ✅ **Project 0**: Loaded your trade data
- ✅ **Project 1**: Discovered trading rules (ran at least one scenario)
- ✅ **Downloaded price data**: XAUUSD H1 candles covering 2022-2024

### Step 2: Run the Backtest ⚡

**Option A - Using UI (Easiest)**:
1. Open the application: `python main_app.py`
2. Click "2 - Backtesting" in sidebar
3. Go to "Run Backtest" panel
4. Click "Run Backtest" button
5. Wait 2-5 minutes
6. Go to "View Results" panel
7. Click "Open HTML Report"

**Option B - Command Line**:
```bash
cd project2_backtesting
python run_backtest.py
```

### Step 3: Interpret Results 📊

Open `outputs/backtest_report.html` and check:

#### ✅ **Good Strategy** (Proceed to Project 3)
- Out-of-sample net profit > 0
- Out-of-sample win rate ≥ 50%
- Out-of-sample profit factor ≥ 1.5
- Max drawdown < 20%
- Similar performance in both periods

#### ⚠️ **Overfitting** (Return to Project 1)
- In-sample great, out-of-sample poor
- Huge performance drop between periods
- Out-of-sample win rate < 40%

#### ❌ **Bad Rules** (Redo Project 1)
- Both periods losing money
- Very low win rate in both periods
- Profit factor < 1.0 in both periods

## ⚙️ Configuration

Edit `backtest_engine.py` if you need to change:

```python
# Date ranges
INSAMPLE_START = '2022-01-01'
INSAMPLE_END = '2023-12-31'
OUTSAMPLE_START = '2024-01-01'
OUTSAMPLE_END = '2024-12-31'

# Capital
STARTING_CAPITAL = 10000.0      # $10,000 starting balance

# Risk
RISK_PER_TRADE_PCT = 0.01       # 1% risk per trade
SL_ATR_MULTIPLIER = 1.5         # Stop loss = ATR × 1.5
TP1_ATR_MULTIPLIER = 1.5        # Take profit 1 = ATR × 1.5 (50% position)
TP2_ATR_MULTIPLIER = 3.0        # Take profit 2 = ATR × 3.0 (50% position)

# Costs
COMMISSION_PER_LOT = 4.0        # $4 per lot (round trip)
SPREAD_PIPS = 0.3               # 0.3 pip spread
```

## 🔧 Troubleshooting

### "Rules file not found"
→ Run Project 1 first: `python run_all_scenarios.py`

### "Price data not found"
→ Download data: `python download_data_mt5.py` (in project1_reverse_engineering)

### "No trades generated"
→ Rules might be too restrictive. Check rules_report.txt conditions.

→ Try opening directly:
- Windows: `start outputs\backtest_report.html`
- Mac: `open outputs/backtest_report.html`

## 💡 Quick Tips

1. **Always check out-of-sample first** - That's the real test
2. **Don't trust in-sample results alone** - They can be misleading
3. **Look for consistency** - Similar performance = robust strategy
4. **Check drawdown** - Even profitable strategies need manageable risk
5. **Review monthly performance** - Avoid strategies that only work some months
